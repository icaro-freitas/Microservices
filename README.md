Projeto desenvolvido no decorrer do Curso  Master Microservices with SpringBoot,Docker,Kubernetes da Eazybytes.

- Spring Boot
- Spring Cloud
- Spring Cloud Functions
- Spring Cloud Stream
- Keycloak
- Docker
- Kubernetes

- MySQL
- Eureka
- Swagger
